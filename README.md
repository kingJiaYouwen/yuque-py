# yuque-py

Python SDK for [yuque](https://www.yuque.com/yuque/developer/api)


## Install

```bash
pip instal yuque-py
```

## Usage

```python
from yuque_py import Yuque

yuque=Yuque(api_host="",user_token="")
yuque.docs
```
